# react-redux-rxjs

My boilerplate code for React + Redux + RxJs

## Install

```sh
npm i
```

## Development

```sh
npm start
```

## Storybook

```sh
npm run storybook
```

## Lint

```sh
npm run lint
```

## Test

```sh
npm test
```

## Commit

```sh
# Please use this script to make a commit because a commit message will be linted by commitlint.
npm run cm
```

## Build a package

```sh
npm run build:comp
```

## Publish packages

```sh
npm run pub
```

## Build

```sh
npm run build
```
